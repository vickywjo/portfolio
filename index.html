About

Vicky is a web developer with a focus in UX/UI Engineering. Outside of web development, she has experience in the performing arts and working in education. While in university, she also volunteered for community causes, and has participated in several case studies as well.

Vicky recieved her B.A. at the University of Central Florida. While knowledgeable in Figma, User-Centered Design, React, React Native, PHP, SQL, JavaScript, CSS, and HTML, she also has experience in working with WordPress, Adobe Photoshop, Adobe Illustrator, GitHub and PHPMyAdmin.

Project & Skill Showcase

Menu App This is a mock restaurant's menu, which accesses all the menu items from an external source using React, rather than being hard coded in. My biggest struggle with this app was getting the menu items to be accurately listed while maintaining the proper styling. After getting it to work with one menu item, I was able to repeat this process for the rest of them.

Quiz App This React Native app allows the users to take a quiz and see their correct and inccrrect answers when done. The users can answer multiple choice, multiple select, or true-or-false questions. Getting the results summary to display with the user's selections visible was difficult as originally, all answer choices would be marked based on what was correct or incorrect. After trial and error I managed to focus on the user's input instead.

FlashCard App Design This app allows users to input flashcard study sets and then use these virtual flashcards to study. The user also has the ability to search for other published study sets if they desire. I wanted this app's design to maintain some similar features as others of the same genre to ensure users would be able to use that knowledge with my design. I also wanted to add more features like being able to add images to each term and quickly flip between cards even without selecting a specific study set.
